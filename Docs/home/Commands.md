*    🖨️ [**Utilities:**](/Docs/utilities.md) `alert` `apply` `break` `bug` `embed` `move`
*    🔨 [**Moderation:**](/Docs/moderation.md) `ban` `unban` `kick` `warn`
*    ✉️ [**Modmail:**](/Docs/modmail.md) `guide` `train`
*    🎮 [**Fun:**](/Docs/fun.md) `emojify`

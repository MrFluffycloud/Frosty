### Commands List 💫 

FA Utilities has a lot of features, with **6 main categories**:

*   👩‍💼 **Information**: `info`, `help`, `ping`, `uptime`! 
*   👻 **Fun**: `emojify`!
*   🖨️ **Utilities**: `alert`, `apply`, `break`, `settings`! 
*   🔨 **Moderation**: `ban`, `unban`, `kick`, `forceban`! 
*   ✉️ **Other**: `serverinfo`, `avatar`!
*   👑 **Owner**: `eval`, `reload`
